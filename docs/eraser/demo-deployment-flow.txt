// Step-by-Step Flow
Client_Browser -> Load_Balancer: 1. Send HTTP request
Load_Balancer -> Demo_App_Container: 2. Route internal traffic
Demo_App_Container -> Identity_Provider: 3. Verify OAuth token
Demo_App_Container -> Redis_Cache: 4. Check cache hit (GET /api/data)
Redis_Cache --> Demo_App_Container: 4a. Return cached data (if hits)
Demo_App_Container -> PostgreSQL_DB: 5. Query data (if cache misses)
PostgreSQL_DB --> Demo_App_Container: 5a. Return database record
Demo_App_Container -> Redis_Cache: 6. Cache retrieved data for 60s
Demo_App_Container --> Client_Browser: 7. Deliver JSON payload to Glassmorphic Client Dashboard
