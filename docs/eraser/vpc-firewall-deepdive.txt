// Inbound Routing Flow
Client -> DNS: Resolve Endpoint
DNS -> Client: Return Public IP
Client -> WAF: Submit HTTPS request
WAF -> Load_Balancer: Inspect & forward clean traffic
Load_Balancer -> Compute_Nodes: Route path/host to private compute nodes

// Egress Routing Flow (Inspected)
Compute_Nodes -> Central_Firewall: Redirect 0.0.0.0/0 outbound traffic
Central_Firewall -> NAT_Gateway: Approve domain whitelist matching
NAT_Gateway -> External_SaaS_APIs: Route request securely via static egress IP
