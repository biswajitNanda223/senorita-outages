Google_Cloud_Trace [icon: monitor, label: "Google Cloud Trace API", color: purple]

// Invocation Flows
Client -> Load_Balancer: POST /api/agent/run
Load_Balancer -> Cloud_Armor: Inspect request
Cloud_Armor -> Agent_Engine: Route traffic (within private bounds)

// Tracing Export (OpenTelemetry)
Agent_Engine -> Google_Cloud_Trace: Export span telemetry (Cloud Trace API)

// Caching and Storage connections
Agent_Engine -> Memorystore_Redis: Fetch/Update chat history (short-term cache)
Agent_Engine -> VPC_Access_Connector: Route storage requests
VPC_Access_Connector -> Storage_Bucket: Save session artifacts & log execution runs
