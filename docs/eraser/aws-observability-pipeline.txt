// Metrics ingestion flow
Container_Pods -> Prom_Exporter: Expose performance data
ADOT_Collector -> Prom_Exporter: Scrape /metrics periodically
ADOT_Collector -> Managed_Prometheus: Ingest metrics using AWS SigV4

// Logs ingestion flow
Container_Pods -> Stdout_Stderr: Emit json log strings
Fluent_Bit -> Stdout_Stderr: Read container log sockets
Fluent_Bit -> CloudWatch_Logs: Ship compressed log streams

// Dashboard visualization flow
Managed_Grafana -> Managed_Prometheus: Query metrics via PromQL
Managed_Grafana -> CloudWatch_Logs: Query text events via CloudWatch APIs
Managed_Grafana -> Operations_Team: Display visual system metrics
