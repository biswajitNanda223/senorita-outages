// Connections
Ingress_LoadBalancer -> Cloud_Armor: Filter Traffic
Cloud_Armor -> GKE_Private_Cluster: Route GKE
Cloud_Armor -> Cloud_Run_Private: Route Serverless

Cloud_Run_Private -> VPC_Connector: VNet Egress
VPC_Connector -> Cloud_SQL_Postgres: Peered Queries
VPC_Connector -> Memorystore_Redis: Peered Cache

GKE_Private_Cluster -> Artifact_Registry: Pull Images
GKE_Private_Cluster -> Secret_Manager: Fetch Secrets
GKE_Private_Cluster -> Cloud_SQL_Postgres: Peered Queries
GKE_Private_Cluster -> Memorystore_Redis: Peered Cache
