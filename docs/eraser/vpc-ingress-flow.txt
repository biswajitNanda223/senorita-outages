// Ingress Routing Flow
Client -> Internet_Gateway: Route traffic
Internet_Gateway -> Load_Balancer: Forward request
Load_Balancer -> Ingress_Controller: Route inside private Node subnet
Ingress_Controller -> ClusterIP_Service: Match hostname & path
ClusterIP_Service -> Pod_1: Load Balance (Round-Robin)
ClusterIP_Service -> Pod_2: Load Balance (Round-Robin)
ClusterIP_Service -> Pod_3: Load Balance (Round-Robin)
