// Log Analytics Workspace
Log_Analytics [icon: monitor, label: "Log Analytics Workspace", color: purple]

// Flows & Peerings
App_Gateway -> Azure_Firewall: Route Inbound
Azure_Firewall -> AKS_Cluster: Route Filtered Inbound
Azure_Firewall -> Container_Apps: Route Filtered Inbound
Azure_Firewall -> App_Service: Route Filtered Inbound

AKS_Cluster -> ACR_Endpoint: Pull Images
AKS_Cluster -> KeyVault_Endpoint: Fetch Secret Keys
AKS_Cluster -> Postgres_Endpoint: DB Queries
AKS_Cluster -> Redis_Endpoint: Read/Write Cache

GitLab_Runner_VM -> ACR_Endpoint: Push Images
GitLab_Runner_VM -> KeyVault_Endpoint: Get Runner Tokens
GitLab_Runner_VM -> Blob_Endpoint: Store Artifacts

// Diagnostics logs pipeline
AKS_Cluster -> Log_Analytics: Stream Audits
KeyVault_Endpoint -> Log_Analytics: Stream AuditEvent
Postgres_Endpoint -> Log_Analytics: Stream Diagnostic Logs
GitLab_Runner_VM -> Log_Analytics: VM Logs
