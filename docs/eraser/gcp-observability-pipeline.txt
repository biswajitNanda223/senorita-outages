// Metrics ingestion flow
Container_Pods -> Prom_Exporter: Expose performance data
GKE_Prom_Collector -> Prom_Exporter: Scrape /metrics periodically
GKE_Prom_Collector -> Monarch_Database: Write metrics via Monarch API

// Logs ingestion flow
Container_Pods -> Stdout_Stderr: Emit console logs
Google_Logging_Agent -> Stdout_Stderr: Capture standard out/err sockets
Google_Logging_Agent -> Cloud_Logging: Stream compressed log events

// Dashboard visualization flow
Grafana_Workspaces -> Monarch_Database: Query metrics via GMP PromQL proxy
Grafana_Workspaces -> Cloud_Logging: Query log events via Cloud Logging APIs
Grafana_Workspaces -> Operations_Team: Display visual system metrics
