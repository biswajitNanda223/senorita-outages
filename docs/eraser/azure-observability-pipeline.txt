// Metrics ingestion flow
Container_Pods -> Prom_Exporter: Expose performance data
Azure_Monitor_Agent -> Prom_Exporter: Scrape /metrics periodically
Azure_Monitor_Agent -> Prometheus_Workspace: Ingest raw time-series metrics

// Logs ingestion flow
Container_Pods -> Stdout_Stderr: Emit json log strings
Fluent_Bit -> Stdout_Stderr: Read container log sockets
Fluent_Bit -> Log_Analytics: Ship compressed audit log entries

// Dashboard visualization flow
Managed_Grafana -> Prometheus_Workspace: Query metrics via PromQL
Managed_Grafana -> Log_Analytics: Query text events via KQL
Managed_Grafana -> Operations_Team: Display visual system metrics
